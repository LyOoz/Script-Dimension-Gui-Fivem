# Script Dimension GUI (FiveM)

A clean and easy-to-use GUI script for managing player dimensions (routing buckets) in FiveM servers.
No more typing commands — control everything through a modern interface.

# Features

- Change player dimension in real-time
- User-friendly GUI
- Instant server sync
- Permission-ready
- Compatible with other resources

# Installation
1. Place the script in your resources folder
2. Add this to your server.cfg

``` bash
ensure Kim_Dimension
```

# Configuration

Edit the config to add zone to change dimension

``` bash
config.lua
```
